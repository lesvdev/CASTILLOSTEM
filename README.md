# Aprende+ · Plataforma Educativa

Sitio educativo para estudiantes colombianos con contenido sobre:

1. **Derechos Humanos** — Historia, artículos clave y contexto colombiano
2. **Inteligencia Emocional** — Modelo de Goleman, barras de competencias, ejercicios
3. **La Célula** — Organelos, teoría celular, división celular
4. **Psicopedagogía** — Vygotsky, Piaget, Gardner, estrategias de aprendizaje
5. **Mundial de Fútbol** — Historia, datos, ciencia y Colombia

## Estructura de archivos

```
/
├── index.html      # Página principal
├── styles.css      # Estilos (paleta pastel verde/amarillo/blanco/negro)
├── script.js       # Interacciones: acordeón, reveal, navbar, animaciones
└── README.md
```

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub (ej: `aprende-plus`)
2. Sube los tres archivos: `index.html`, `styles.css`, `script.js`
3. Ve a **Settings → Pages**
4. En **Source**, selecciona la rama `main` y la carpeta `/root`
5. Guarda. En unos minutos el sitio estará en:
   `https://TU-USUARIO.github.io/aprende-plus/`

## Características

- Diseño responsive (móvil, tablet, desktop)
- Animaciones de scroll reveal
- Acordeón interactivo
- Barras de progreso animadas
- Marquee band
- Tilt effect en las tarjetas
- Easter egg: código Konami para modo arcoíris 🌈
